Справочник API
==============

Открытие сессии
---------------
Для начала работы необходимо открыть клиентскую сессию поддерживающую пул соединений с MOEX ISS в режиме keep alive.
При осуществлении множества запросов рекомендуется иметь одну клиентскую сессию в течении всей работы приложения для
ускорения взаимодействия с сервером.
В конце работы для высвобождения ресурсов сессия должна быть закрыта вызовом метода closed
